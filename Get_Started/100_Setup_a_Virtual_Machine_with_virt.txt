====== Setup a Virtual Machine with virt ======
Created Saturday 30 January 2016
	a. use wget to downloadde iso and move (mv) to /tmp 
	
http://mirror0.babylon.network/centos/7/isos/x86_64/CentOS-7-x86_64-Minimal-1511.iso 

		virt-install --name=node1.controler --disk path=/var/lib/libvirt/images/vm1.img,size=12 --ram=512 --os-type=linux --os-variant=rhel6 --network bridge:br0 --nographics --cdroom=/temp/Centos-6.3-x86_64-minimal.isono
		
		 IMPORTANT: do not hit "ENTER" use "TAB" to modify instalation options
		type
		 > vmlinuz initrd.img console=ttyS0,115200
